编译安装OpenSSL

用了好久的OpenSSL，都没有写篇博客来介绍一下

OpenSSL是一个开源的加密库，提供了https所需要的SSL层支持。

在ubuntu系统上可以直接使用sudo apt-get install openssl进行安装，但是由于版本原因，建议直接官网下载所需要的版本进行编译安装

下载OpenSSL

下载请移步 https://www.openssl.org/source/

选择你所需要的版本进行下载

编译

编译的时候可以直接输入make,这样的话很多openssl的配置属性是不会有的，很纯，但性能不行。

所以建议使用命令 ./configure 这样会生成配置文件。

然后再使用命令 make 就会进入编译时间，等一会，就编译完成了。

安装

编译完成后直接输入命令 make install 便完成安装，

接下来需要修改PATH，将openssl的路径添加进去。
